# Segundo Desafio IUUL

No segundo desafio da IUUL, nós desenvolvemos uma maneira de deixar mensagens no portfólio utilizando o formulário de contato.

## Aplicação das Heurísticas de Nielsen

Na minha solução, utilizei a heurística número 5 (Prevenção de erros) de acordo com esse [site](https://brasil.uxdesign.cc/10-heur%C3%ADsticas-de-nielsen-para-o-design-de-interface-58d782821840).

<img src="https://i.ibb.co/nfRTptq/a.png" alt="a" border="0">

Ou seja, quando o usuário clica no botão para excluir as mensagens, uma pequena janela com dois botões é aberta e aguarda a confirmação do usuário para finalmente excluir as mensagens, desse modo prevenindo a exclusão das mensagens por engano.

A segunda heurística usada foi a número 1 (Visibilidade do Status do Sistema), deixando o usuário sempre atualizado com o que está acontecendo, ou o que aconteceu com a mensagem que ele tentou enviar. Por exemplo, caso ele tente enviar e ocorra um erro:

Caso tenha sucesso:
